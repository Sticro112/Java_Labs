# лабораторна робота 1

## Опис

Ознайомитись з порядком створення нового проекту та написати першу програму.

## Вимоги

- **JDK** : версія 25 і вище
- **Git** : для клонування репозиторію

## Архітектура

* **Main.java** — Головна точка входу в програму. Ініціалізує загальний об'єкт `Scanner`, послідовно викликає статичні методи `execute()` кожного завдання та забезпечує коректне закриття потоків введення по завершенню роботи.
* **SafeEngine.java** — Допоміжний сервісний клас для безпечного зчитування даних з консолі. Запобігає аварійному завершенню програми (`Crash-proof`) у разі введення користувачем некоректних типів даних (наприклад, літер замість цифр), використовуючи конструкцію `try-catch` та обробку винятку `InputMismatchException`.
* **Task1.java** — Клас для математичного аналізу числового масиву з 10 елементів. Обчислює суму, різницю, добуток та частку елементів (із захистом від ділення на нуль).
* **Task2.java** — Робота з текстовими масивами. Зчитує 10 окремих слів від користувача та динамічно конструює з них цілісне речення.
* **Task3.java** — Робота з масивами користувацьких об'єктів. Створює масив екземплярів внутрішнього класу `User` (поля: `id`, `age`, `name`, `surname`, `weight`, `height`) та обчислює їхні середні статистичні показники.
* **Task4.java** — Робота з об'єктами сутностей. Створює масив моделей `Car` (поля: `hp`, `volume`, `model`, `brand`, `color`, `year`, `isNew`) та підраховує їхні сумарні технічні показники (потужність та об'єм).
* **Task5.java** — Алгоритм математичного реверсу. Виконує порозрядне розщеплення тризна

Проєкт написаний на чистій Java без використання сторонніх залежностей або систем збирання (Maven/Gradle), тому легко запускається через стандартний CLI.

### Вимоги:
* Встановлений **JDK** версії 8 або вище.

### Інструкція із запуску:
1. Відкрий термінал або командний рядок у папці із сирцевим кодом (де знаходяться файли `.java`).
2. Скомпілюй усі класи разом:
   ```bash
   javac Main.java SafeEngine.java Task*.java
