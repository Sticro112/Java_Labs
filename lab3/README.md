# Лабораторна робота: Вивчення базових алгоритмічних структур мови Java

Цей проєкт є консольним Java-додатком, розробленим для демонстрації та практичного застосування базових керуючих конструкцій мови Java (цикли `for`, `while`, вкладені цикли), а також механізмів обробки винятків для безпечного введення даних користувачем.

## 📌 Структура проєкту

Проєкт розділений на окремі класи відповідно до принципу єдиної відповідальності (**SRP**):

* **`Main.java`** — Головна точка входу в програму. Координує послідовний запуск усіх завдань.
* **`safeEngine.java`** — Утилітарний клас (рушій) для безпечного зчитування чисел з консолі. Захищає програму від падіння при помилковому введенні тексту замість цифр.
* **`Task1.java`** — Демонстрація найпростіших циклів `for` та `while` (виведення текстової фрази 50 разів з нумерацією).
* **`Task2.java`** — Реалізація лічильника часу (години та хвилини) за допомогою вкладених циклів `for`.
* **`Task3.java`** — Аналогічний лічильник часу (години та хвилини), реалізований за допомогою вкладених циклів `while`.
* **`Task4.java`** — Трьохрівневий вкладений цикл `for` для повної симуляції електронного годинника (години, хвилини, секунди).
* **`Task5.java`** — Обчислення значень математичної функції $y = 2 \cdot \cos(x) + 3$ на інтервалі $[1.0; 3.0]$ з кроком $0.1$. Виконує підрахунок кількості значень, що потрапили в заданий користувачем діапазон $[y_{min}; y_{max}]$.

---

## 🛠️ Функціонал безпечного введення (`safeEngine`)

Для того щоб програма не завершувала роботу аварійно через `InputMismatchException` (наприклад, якщо користувач ввів літери замість чисел), реалізовано методи перехоплення помилок:
* `readDouble(Scanner scan, String prompt)` — циклічно запитує дробове число, використовуючи локалізацію з комою `,` як роздільником.
* `readInt(Scanner scan, String prompt)` — циклічно запитує ціле число.
