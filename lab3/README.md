# Лабораторна робота: Вивчення базових алгоритмічних структур мови Java

Цей проєкт є консольним Java-додатком, призначеним для вивчення, аналізу та практичного застосування базових керуючих конструкцій мови Java, вкладених циклів різного типу, а також механізмів перехоплення винятків для організації безпечного введення даних.

## 1. Структура проєкту

Проєкт спроєктовано із розподілом логічних блоків за окремими класами:

* Main.java — Головна точка входу в програму. Містить метод main, який координує послідовний виклик і виконання всіх завдань лабораторної роботи.
* safeEngine.java — Утилітарний клас, що реалізує логіку безпечного зчитування числових даних з консолі. Запобігає аварійному завершенню програми у разі некоректного введення користувачем текстових символів.
* Task1.java — Реалізація циклічних конструкцій for та while для виведення текстового рядка задану кількість разів (50 ітерацій) з виведенням поточного номера кроку.
* Task2.java — Симуляція лічильника часу (години та хвилини) в межах заданого часового інтервалу з використанням вкладених циклів типу for.
* Task3.java — Симуляція лічильника часу (години та хвилини) за аналогічним алгоритмом, але реалізована за допомогою вкладених циклів типу while.
* Task4.java — Розширена трьохрівнева модель електронного годинника, яка додатково враховує секунди, побудована на базі вкладених циклів for.
* Task5.java — Табулювання математичної функції y = 2 * cos(x) + 3 на проміжку [1.0; 3.0] з кроком 0.1. Клас виконує обчислення значень через цикли for та while, здійснює порівняльний аналіз результатів та підраховує кількість значень y, що задовольняють заданий користувачем діапазон [y_min; y_max].

## 2. Механізм валідації даних (safeEngine)

Для запобігання винятковим ситуаціям типу InputMismatchException, які виникають при спробі зчитати нечислове значення через Scanner, вхідні потоки загорнуті в блоки перехоплення винятків:
* readDouble(Scanner scan, String prompt) — зчитує дробові числа у нескінченному циклі доти, доки не буде введено валідне значення (враховує регіональні налаштування системи, де роздільником є кома).
* readInt(Scanner scan, String prompt) — забезпечує аналогічний циклічний запит для цілих чисел типу int.

## 3. Компіляція та запуск програми

Для запуску додатку необхідно мати встановлений пакет розробника JDK (Java Development Kit) версії 11 або вище.

Виконання команд у терміналі:
1. Компіляція всіх сирцевих файлів проєкту:
   javac *.java

2. Запуск скомпільованого байт-коду:
   java Main
